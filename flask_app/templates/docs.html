Quick Start

# Search manga
python cli.py search "One Piece"

# Download all chapters as EPUB (default)
python cli.py download --id <manga_id> --source mangadex

# Download specific chapters as CBZ
python cli.py download --id <id> --chapters 1,2,3 --format cbz

# Bulk download from file
python cli.py bulk --file urls.txt --concurrent 3

Commands

search <query>

Search manga across sources. Flags: --sources, --rating, --limit.

python cli.py search "Berserk"
python cli.py search "Solo Leveling" --sources mangadex,asurascans

info [--id] [--url] [--source] [--list]

Get manga details, description, tags, chapters, volumes.

python cli.py info --id abc-123 --source mangadex --list

download [options]

Download with full control over chapters, volumes, format, split or single file.

python cli.py download --id abc-123 --format epub --split
python cli.py download --id abc-123 --chapters 1,2,3 --format cbz
python cli.py download --id abc-123 --volumes 1,2 --crop --quality 90
python cli.py download --id abc-123 --no-split --format epub
python cli.py download --url "https://mangadex.org/title/abc-123"

bulk --file | --urls

Bulk download from a file or comma-separated URLs.

python cli.py bulk --file urls.txt --format epub --concurrent 5

Notes

  • Cropping is disabled by default — full-size images
  • Default format: EPUB
  • Default rating: any (all content levels)
  • EPUB files include embedded metadata (title, author, cover art)
  • Naming variables: {title}, {chapter}, {volume}, {source}
