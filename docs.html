Quick Start

# Search manga
python cli.py search "One Piece"

# Download all chapters as EPUB (default format)
python cli.py download --id <manga_id> --source mangadex

# Download specific chapters as CBZ
python cli.py download --id <id> --chapters 1,2,3 --format cbz

# Download by URL
python cli.py download --url "https://mangadex.org/title/..."

# Bulk download from file
python cli.py bulk --file urls.txt --concurrent 3

# Launch interactive TUI
python tui.py

# Launch desktop GUI
python gui.py

# Launch web server
python flask_app/app.py

Commands

search <query>

Search manga across enabled sources. Accepts --sources, --rating, and --limit flags.

# Basic search across all sources
python cli.py search "Berserk"

# Search specific sources only
python cli.py search "Solo Leveling" --sources mangadex,asurascans

# Filter results by content rating
python cli.py search "Naruto" --rating safe

# Increase result limit
python cli.py search "One Piece" --limit 50

info [--id] [--url] [--source] [--list]

Retrieve detailed manga information including description, tags, chapter and volume lists.

# Get info by ID
python cli.py info --id abc-123-def --source mangadex

# Get info by URL
python cli.py info --url "https://mangadex.org/title/abc-123-def"

# Show all available chapters
python cli.py info --id abc-123 --source mangadex --list

download [--id] [--url] [--source] [--chapters] [--volumes] [--split/--no-split]

Download manga with full control over what and how to download. Supports chapter ranges, volume selection, split or single-file output.

# Download everything as EPUB (default)
python cli.py download --id abc-123 --source mangadex

# Download as CBZ with high quality
python cli.py download --id abc-123 --format cbz --quality 95

# Download specific chapters (comma-separated or range)
python cli.py download --id abc-123 --chapters 1,2,3,5-10

# Download full volumes
python cli.py download --id abc-123 --volumes 1,2

# Combine all chapters into a single file
python cli.py download --id abc-123 --no-split --format epub

# Download from a URL
python cli.py download --url "https://mangadex.org/title/abc-123"

# Enable cropping with custom quality
python cli.py download --id abc-123 --crop --quality 90

# Specify output directory
python cli.py download --id abc-123 -o "/path/to/manga"

bulk --file | --urls

Process multiple URLs from a file or comma-separated list. Each URL is parsed and downloaded automatically.

# Bulk from file (one URL per line)
python cli.py bulk --file urls.txt --format epub --concurrent 5

# Bulk from command line
python cli.py bulk --urls "url1,url2,url3" --format cbz

# Bulk with language filter
python cli.py bulk --file manga_list.txt --format epub --lang en

Advanced Usage

Naming Schemes

Custom output filenames using variables: {title}, {chapter}, {volume}, {source}

# Default naming
{title}_ch{chapter}.epub

# Volume and chapter prefix
{title}_Vol.{volume}_Ch.{chapter}.cbz

# Source-prefixed naming
[{source}] {title} - Chapter {chapter}.epub

Metadata

EPUB files include embedded metadata: title, author, cover art, and publisher. Use the GUI or web interface to edit metadata fields before downloading.

Rating Filtering

# Include all content ratings (default)
--rating any

# Safe content only
--rating safe

# Specific levels
--rating suggestive
--rating erotica
--rating pornographic

Notes

  • Cropping is disabled by default — all images are saved at full original size
  • Default output format is EPUB
  • Default content rating is any (includes safe, suggestive, erotica, and pornographic)
  • EPUB files include embedded metadata: title, author, cover art
  • Concurrent downloads speed up bulk operations (default: 3)
  • Naming scheme variables: {title}, {chapter}, {volume}, {source}
  • Install all dependencies: pip install -r requirements.txt
